โปรแกรมบัญชีธนาคารเพื่อการทำธุรกรรมเบื้องต้น เขียนด้วยภาษา C# 

	เป็นโปรแกรมที่จำลองการทำธุรกรรมทางการเงินในบัญชีธนาคาร โดยผู้ใช้สามารถทำธุรกรรมพื้นฐาน ได้แก่ การฝากเงิน การถอนเงิน 
และการตรวจสอบยอดเงินคงเหลือในบัญชีผ่าน Console Application

ความสามารถของโปรแกรม
- บัญชีจำเป็นต้องมีเงินในบัญชีอย่างน้อย 100 บาท จึงจะทำการทำธุรกรรมใด ๆ ได้
- รองรับการทำธุรกรรมพื้นฐาน ได้แก่
	1. Deposit (การฝากเงิน) ผูัใช้สามารถฝากเงินเข้าบัญชีได้ โดยกรอกจำนวนเงินที่ต้องการฝาก ระบบจะอัปเดตยอดเงินคงเหลือในบัญชีทันที
	โดยเงินในบัญชีจะต้องมีเงินคงเหลือ 100 บาทขึ้นไปเพื่อให้สามารถทำการฝากเงินได้
	2. Withdraw (การถอนเงิน) ผู้ใช้สามารถถอนเงินจากบัญชีได้ โดยกรอกจำนวนเงินที่ต้องการถอน ระบบจะตรวจสอบยอดเงินคงเหลือในบัญชีก่อน 
	หากยอดเงินเพียงพอ ระบบจะดำเนินการถอนเงินและอัปเดตยอดเงินคงเหลือในบัญชีทันที หากยอดเงินไม่เพียงพอ ระบบจะแจ้งเตือนผู้ใช้โดยบัญชีจำเป็นต้อง
	มีเงินคงเหลือ 100 บาทขึ้นไปเพื่อให้สามารถทำการถอนเงินได้
	3. Check Balance (การตรวจสอบยอดเงินคงเหลือ) ผู้ใช้สามารถเช็คยอดเงินคงเหลือในบัญชีได้ตลอดเวลา โดยระบบจะแสดงยอดเงินคงเหลือในบัญชีให้ผู้ใช้ทราบ
- รองรับการทำธุรกรรมหลายครั้งในหนึ่งรันของโปรแกรม ผู้ใช้สามารถทำธุรกรรมได้ต่อเนื่องโดยไม่ต้องออกจากโปรแกรม

วิธีการใช้งาน
1. รันโปรแกรมผ่าน Console Application
2. ผู้ใช้ฝากเงินแรกเข้า บังคับการมีเงินในบัญชีอย่างน้อย 100 บาท จึงจะทำธุรกรรมใด ๆ ได้
3. ผู้ใช้จะได้รับเมนูให้เลือกทำธุรกรรม ได้แก่ (1)Deposit, (2)Withdraw, (3)Check Balance และ (4)Exit
4. ผู้ใช้เลือกทำธุรกรรมที่ต้องการ โดยกรอกตัวเลขที่ตรงกับเมนู หากกรอกข้อมูลไม่ถูกต้องโปรแกรมจะทำการแสดงข้อความแจ้งเตือนต่อผู้ใช้
5. กดปุ่มใด ๆ เพื่อดำเนินการตามที่เลือกได้เรื่อย ๆ จนกว่าจะเลือก (4)Exit เพื่อออกจากโปรแกรม
