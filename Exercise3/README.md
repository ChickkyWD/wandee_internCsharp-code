โปรแกรม Hangman Game เขียนด้วยภาษา C#

   โปรแกรมนี้เป็นเกม Hangman สร้างขึ้นเพื่อให้ผู้เล่นทายคำศัพท์ โดยผู้เล่นจะต้องทายคำศัพท์ที่ถูกสุ่มขึ้นมาโดยโปรแกรม โดยมีการแสดงจำนวนตัวอักษรในคำศัพท์และจำนวนครั้งที่ผู้เล่นสามารถทายผิดได้ก่อนที่จะเสียชีวิต

ความสามารถของโปรแกรม
- รองรับการเล่นเกม Hangman โดยมีฟังก์ชันได้แก่ 
	1. การเริ่มเกมใหม่ ผู้เล่นสามารถเริ่มเกมใหม่ได้โดยการกด "Enter" ซึ่งจะสุ่มคำศัพท์ใหม่และรีเซ็ตสถานะของเกม
	2. การทายตัวอักษร โดยผู้เล่นกรอกตัวอักษรทีละ 1 ตัว จากนั้นโปรแกรมจะทำการเช็คสถานะตัวอักษร ได้แก่
		ถูก แสดงตัวอักษรในตำแหน่งที่ถูกต้อง
		ผิด ลดจำนวนครั้งที่เหลือ จาก 10 ครั้ง
		ซ้ำ แจ้งเตือนผู้เล่นว่า "You have already tried this character."
	3. และการแสดงผลลัพธ์ของเกม 
		ชนะ แสดงข้อความ "Congratulation, you're win!"
		แพ้ แสดงข้อความ "Game Over! You lose."

วิธีการใช้งาน
1. รันโปรแกรมผ่าน  Console Application
2. โปรแกรมจะสุ่มคำศัพท์จากรายการ (Word List)
3. ผู้เล่นสุ่มตัวอักษรภาษาอังกฤษทีละ 1 ตัว เพื่อให้โปรแกรมเช็คเงื่อนไขสถานะของตัวอักษร
4. หากผู้เล่นชนะ หรือแพ้ หรือกด Enter โดยไม่กดตัวอักษรใด ๆ โปรแกรมจะทำการเริ่มเกมใหม่
