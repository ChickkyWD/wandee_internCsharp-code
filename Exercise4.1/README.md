โปรแกรมคำนวณพื้นที่ผิวกล่อง เขียนด้วยภาษา C#

   โปรแกรมนี้เป็นการจำลองกล่องสี่เหลี่ยม โดยมีฟังก์ชันได้แก่ การสร้างกล่องสี่เหลี่ยม การคำนวณพื้นที่ ผู้ใช้สามารถทำการจำลองกล่องสี่เหลี่ยมได้ผ่าน Console Application

ความสามารถของโปรแกรม
- รองรับการสร้างกล่องสี่เหลี่ยม 2 แบบ ได้แก่
	1. กล่องสี่เหลี่ยมทั่วไป ผู้ใช้ต้องความกว้าง ความยาว และความสูงของกล่องเพื่อสร้างกล่องสี่เหลี่ยมทั่วไป
	2. กล่องสี่เหลี่ยมลูกบาศก์ ผู้ใช้ต้องความยาวด้านเดียวเพื่อสร้างกล่องสี่เหลี่ยมลูกบาศก์
	3. การคำนวณพื้นผิวแต่ละด้าน และพื้นที่ของกล่องสี่เหลี่ยม โดยระบบจะคำนวณและแสดงผลลัพธ์ให้ผู้ใช้ทราบ
	4. โปรแกรมทำการคัดลอกกล่องสี่เหลี่ยมที่สร้างขึ้นมาใหม่ได้

วิธีการใช้งาน
1. รันโปรแกรมผ่าน Console Application
2. ผู้ใช้จะได้รับเมนูให้เลือกสร้างกล่องสี่เหลี่ยม ได้แก่ (1)กล่องสี่เหลี่ยมทั่วไป, (2)กล่องสี่เหลี่ยมลูกบาศก์ 
3. โปรแกรมรับค่าจากผู้ใช้
4. โปรแกรมคำนวณพื้นที่ผิวแต่ละด้าน และพื้นที่ผิวรวม
5. โปรแกรมคัดลอกกล่องที่สร้างขึ้นมาเป็นกล่องใหม่ที่มีขนาดเท่ากัน
5. ดูผลลัพธ์ใน Console Application
