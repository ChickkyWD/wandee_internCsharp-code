โปรแกรมจำลองการทอยลูกเต๋า เขียนด้วยภาษา C#

   โปรแกรมนี้เป็นการจำลองการทอยลูกเต๋า โดยมีฟังก์ชันได้แก่ การทอยลูกเต๋า การแสดงผลลัพธ์ของลูกเต๋าที่ทอย และการนับจำนวนครั้งที่ทอยลูกเต๋า ผู้ใช้สามารถทำการจำลองการทอยลูกเต๋าได้ผ่าน Console Application

ความสามารถของโปรแกรม
- รองรับการทอยลูกเต๋าซึ่งจะสุ่มตัวเลขระหว่าง 1 ถึง 6 และแสดงผลลัพธ์ของลูกเต๋าที่ทอย 2 ส่วนหลักคือ
	1. ทอยลูกเต๋าจนกว่าจะได้เลข 1
	2. ทอยลูกเต๋า 2 ลูกจนได้ค่า (1,1) หรือ Snake Eyes
	3. นับจำนวนครั้งที่ทอยลูกเต๋าในแต่ละกรณี และแสดงผลลัพธ์ให้ผู้ใช้ทราบ

วิธีการใช้งาน
1. รันโปรแกรมผ่าน Console Application
2. โปรแกรมจะทำการทอยลูกเต๋าโดยอัตโนมัติและแสดงผลลัพธ์ของลูกเต๋าที่ทอยในแต่ละรอบ
3. โปรแกรมจะนับจำนวนครั้งที่ทอยลูกเต๋าในแต่ละกรณีและแสดงผลลัพธ์ให้ผู้ใช้ทราบเมื่อได้เลข 1 และค่า (1,1) หรือ Snake Eyes
4. เสร็จสิ้นการทอยลูกเต๋าและแสดงผลลัพธ์ทั้งหมดใน Console Application
