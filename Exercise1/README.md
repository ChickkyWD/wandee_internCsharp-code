โปรแกรมใช้สำหรับคำนวณการทอนเงิน เขียนด้วยภาษา C#

 รันผ่าน Console Application ผู้ใช้สามารถกำหนดจำนวนแบงค์ และเหรียญเริ่มต้นได้ จากนั้นระบบจะคำนวณการทอนเงิน และแสดงค่าแบงค์ 
และเหรียญว่าต้องใช้จำนวนเท่าใดบ้าง และคำนวณส่วนที่เหลือในคลังออกมา

ความสามารถของโปรแกรม
- รองรับหน่วยเงินเป็นแบงค์ 1000 500 100 50 และเหรียญ 10 5 1
- คำนวณการทอนเงินอัตโนมัติ
- ตรวจสอบการทอนเงิน 3 กรณี ได้แก่
	1. จำนวนเงินเพียงพอสำหรับการทอนเงิน
	2. เงินไม่พอสำหรับการทอนเงิน คือจำนวนเงินในคลังมีน้อยกว่าจำนวนเงินที่ต้องการทอนเงิน
	3. ไม่มีเศษสำหรับการทอนเงิน คือจำนวนเงินในคลังไม่มีเศษย่อยเพียงพอต่อการทอนเงิน
- แสดงจำนวนเงินที่ใช้ และจำนวนเงินคงเหลือในคลังในการทอนเงินแต่ละครั้ง

วิธีการใช้งาน
1. รันโปรแกรมผ่าน Console Application
2. ผู้ใช้กรอกจำนวนแบงค์ และเหรียญเริ่มต้น
3. ผู้ใช้กรอกจำนวนเงินทีต้องการทอน
4. ระบบคำนวณโดยใช้แบงค์ และเหรียญมูลค่าสูงสุดก่อน ลดหลั่นตามลำดับ
5. หากทอนได้ จะแสดงรายการแบงค์ และเหรียญที่ใช้ อัปเดตจำนวนเงินคงเหลือในคลัง
6. หากทอนไม่ได้ จะแสดงการแจ้งเตือนต่อผู้ตามการตรวจสอบกรณีที่ 2 และ 3
7. โปรแกรมวนลูปต่อไปเรื่อย ๆ
